## 適用手順

1. このリポジトリを「GitHub Pages」として公開する（＝[このページ](#GitHub-Pages)）。

2. はてなブログ > 設定 > 詳細設定 > ヘッダーへ、下記を入力。
```html
<link rel="stylesheet" href="//yuricks7.github.io/HatenaBlogCustmize/css/custmize.css">
```
※出来るだけ下の方へ。

3.  表示に問題がなければ、「変更」をクリック。
